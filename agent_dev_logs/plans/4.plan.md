## Strategic actions

### Phase 0.1 тАФ Commit the missing design doc (resolve dangling refs) тАФ DO FIRST
`docs/translation-backends.md` is cited 5+ times in shipped code but absent on `origin/test`.
Commit it (its full content is in the issue thread): the backend contract, the resolution order,
the re-ranked candidate table, the **full licensing recipe** (incl. FastText langid + NC vocab),
and the eval plan. Until this lands, `tests/test_backend.py` references and the module docstrings
point at a non-existent file.

### Phase 2 тАФ Run the bake-off (answer the original question)
This is the deliverable the issue title actually asks for. Run `eval/bakeoff.py`:
1. CUBBITT vs `openai_compatible` (┬▒ glossary) on a sample of **AMCR metadata + ALTO OCR**
   (`data_samples/`), `--limit ~40`.
2. Metrics: **chrF/BLEU** (sacrebleu) + **COMET** where refs exist; **COMET-QE / LLM-judge +
   heuristic QE** (number/date/code preservation, empty-output rate, length-ratio) for the common
   reference-less archival case; **terminology hit-rate** from the vocab CSV; **OCR-robustness**
   spot check (does the backend hallucinate/"correct" garbled tokens?); pairwise divergence.
3. Output the per-segment + summary CSV; **decide default vs opt-in** from results and record it.

### Phase 1.1 тАФ Live free-LLM smoke (cheap, high-signal)
Run one real translation through `--backend openai_compatible` with a free key (OpenRouter free
model) via **both** CLI and the `/translate` API, on a sample ALTO + metadata file тАФ confirms the
guards (temp 0, length-ratio reject) and the prompt-glossary path behave on real OCR, not just
mocks.

### Phase 3 тАФ Permissive self-host (after the bake-off picks a direction)
If NC licensing or offline deployment becomes the driver, finish `ct2_translator.py`:
- Register `ct2` (uncomment the two lines), convert **EuroLLM-1.7B** and/or **MADLAD-400-3B** to
  CT2 int8/int4, validate the faithfulness guards (`_MIN/_MAX_LEN_RATIO`, `max_decoding_length`).
- Execute the **permissive recipe** end-to-end: `--backend ct2` (Apache-2.0 model) **+** explicit
  `--source_lang` (no FastText CC-BY-NC) **+** permissive/empty vocab тЖТ verify `para_licenses`
  computes a permissive output license from the logged components.
- Pin the real `llm_api` license terms (currently the deliberate maximally-restrictive placeholder).
