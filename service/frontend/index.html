ATRIUM Translator 🌍

Upload an ALTO XML page or an AMCR metadata XML file to translate its text content. The translated XML is returned as a download.

Translating, please wait (long documents take minutes)...

API usage 📡

This page is a thin client over the REST API. The same call from the command line:

curl -X POST "http://localhost:8000/translate?source_lang=auto&target_lang=en&is_alto=true" \
  -F "file=@page.alto.xml" -OJ
